Block Image
Syntax

A line solely containing an image link, preceded by a blank line.

Example
AMDL
Copy
This is a block image:

![Sandcastle](Castle.png "Pirate Sandcastle")
Example Output

This is a block image:

Sandcastle

Notes

The link text will be treated as the image's caption. Omitting the link text will result in a captionless image.

amdl2maml Support

Completed, save for alt text.
