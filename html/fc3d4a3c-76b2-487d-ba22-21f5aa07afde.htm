AMDL to MAML Converter

This topic describes the AMDL to MAML Converter, a.k.a. amdl2maml.

Goal

Create a shim to feed AMDL conceptual topics into existing tools (i.e., SHFB).

About

This site is built (semi-automatically) from AMDL.

amdl2maml uses a fork of CommonMark.NET.

It also feeds the documentation project, since SHFB won't build one with no Documentation Sources.

Status

At the moment amdl2maml supports the following AMDL syntax elements:

  1. Sections and subsections from both ATX-style and Setext-style headers.

  2. Topic and section outlines from Setext-Style Headers.

  3. Topic summaries and introductions.

  4. Fenced, indented and inline code.

  5. Ordered, bulleted and unbulleted lists.

  6. Paragraphs, soft line breaks and quotes.

  7. Inline conceptual, internal and external links.

  8. See Also sections from references and crafted sections.

  9. Emphasis and Strong Emphasis.

  10. Subscript and Superscript.

  11. Block and inline images.

  12. Block and inline HTML.

The following is an incomplete list of missing elements:

  1. Reference link.

  2. Table.

  3. Definition List.

Ideally, amdl2maml should be able to produce an SHFB project without human intervention. Right now it can only read/write the content layout XML.
