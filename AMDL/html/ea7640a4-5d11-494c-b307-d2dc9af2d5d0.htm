Strikeout

This topic details the AMDL syntax of strikeout.

Syntax

N/A.

Motivation

  1. It is not standard Markdown.

  2. No MAML counterpart is known to exist.

  3. A trivial workaround exists, see Workaround.

amdl2maml Support

Any content enclosed in double tildes is interpreted as subscript within subscript.

Example
AMDL
Copy
~~Any content enclosed in double tildes is interpreted as subscript within subscript.~~
Example Output

Any content enclosed in double tildes is interpreted as subscript within subscript.

Workaround
Copy
This element will be <s>deleted</s>
