How To Contribute to the AMDL Documentation

This topic details the steps necessary for contributing to the AMDL documenation.

The AMDL documentation is hosted on GitHub. Basic git knowledge and a GitHub account are prerequisites.

Contributing to the AMDL documentation

  1. Read and understand this documentation.

  2. Read and understand the terms and conditions of the CC-BY-SA 4.0 License.

  3. Obtain your copy of the documentation:

    1. Fork the AMDL repository.

    2. Clone your fork of AMDL.wiki.git.

  4. Add a feature or fix a bug.

  5. Test thoroughly.

  6. Submit your changes:

    1. Commit and push.

    2. Create a pull request.
