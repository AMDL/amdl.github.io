Function

Embed an image in a document.

Syntax

A link immediately preceded by a ! character; formalized syntax TBD.

An image appearing as the sole entity in its block should be interpreted as a block image; otherwise, it should be interpreted as an inline image.

The link text of a block image should be interpreted as the image's caption. If no link text is specified, a captionless image should be generated.

Examples

1. Inline image 1:

AMDL
Copy
This is an inline image:
![](Castle.png "Pirate Sandcastle")

2. Inline image 2:

AMDL
Copy
![](Castle.png "Pirate Sandcastle")
is also an inline image.

3. Block image:

AMDL
Copy
This is a block image:

![Sandcastle](Castle.png "Pirate Sandcastle")
