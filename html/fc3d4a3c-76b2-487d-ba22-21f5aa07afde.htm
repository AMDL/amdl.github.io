Amdl.Maml.Converter
AMDL to MAML Converter

This topic describes the AMDL to MAML Converter, a.k.a. amdl2maml.

Goal

Provide a means of feeding AMDL topics into existing tools (i.e., SHFB).

About

This site is built (semi-automatically) from AMDL.

Dependencies

A build requires the following:

  1. A forked CommonMark.NET.

  2. A forked PCLStorage.

Status

Topic Types

The following topic types are currently supported:

  1. Empty topic.

  2. General topic.

  3. Glossary topic.

  4. How-To topic.

  5. Orientation topic.

Entities

The following entities are currently partially or fully supported:

  1. Sections and subsections, including See Also.

  2. Topic and section outlines from Setext-style headers.

  3. Topic summaries and introductions.

  4. Paragraphs, quotes and soft and hard line breaks.

  5. Ordered, bulleted and unbulleted lists.

  6. Fenced, indented and inline code.

  7. Inline conceptual, internal and external links.

  8. Block and inline images.

  9. Block and inline HTML.

  10. Emphasis and Strong Emphasis.

  11. Subscript and Superscript.

The following is an incomplete list of missing entities:

  1. Reference link.

  2. Table.

  3. Definition List.

Ideally, amdl2maml should be able to produce an SHFB project without human intervention. Right now it can only read/write the content layout XML.
