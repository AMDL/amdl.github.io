ATX-Style Section Header

This topic details the syntax and the AMDL semantics of the ATX-style section header.

Function

An ATX-style section header serves the following purposes:

  1. Assign the section's title with the header's text.

  2. Signify the termination of the topic's introduction, if present.

  3. Signify the opening of a new section.

Syntax

Two to six # signs and a line of text, optionally followed by any number of # signs.
