amdl2maml: Usage

amdl2maml -s sourcePath -d destinationPath -l contentLayoutPath

  • sourcePath

    • Source folder path

  • destinationPath

    • Destination folder path

  • contentLayoutPath

    • Content layout file path

  • -v, --Verbosity

    • Verbosity, one of the following:

      • silent

      • No output

      • minimal

      • Minimal output

      • normal

      • Normal output (the default)

      • detailed

      • Detailed output

      • insane

      • Very detailed output

  • -tf, --TimeFormat

    • Time format (The default is yyyy-MM-dd HH:mm:ss.fff)

  • -df, --DurationFormat

    • Duration format (The default is hh:mm:ss.fff)

  • -?, --Help

    • Display this help
